# Flowzero Releases

This repository hosts the binary releases for [Flowzero](https://flowzero.app) - a speech-to-text application optimized for Chinese.

## Download

Visit the [Releases](https://github.com/daymade/flowzero-releases/releases) page to download the latest version.

## Channels

- **Stable**: Production-ready releases
- **Beta**: Pre-release versions (tagged with `-beta.x`)

## Update Server

Flowzero uses automatic updates via Hazel. The update server reads releases from this repository.

- Stable: `https://updates.flowzero.app`
- Beta: `https://updates-beta.flowzero.app`

## Security

All releases are:
- Code-signed with Apple Developer ID
- Notarized by Apple
- Built from the private source repository via GitHub Actions
